## 高校科研管理系统

> [点我下载源码或在线试用](https://www.notmaker.com/detail/d58f03498e3a4b668e7d41ae05b5efdd/ghb20250810) 

> 支持定制、讲解、修改、远程调试

### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：教师、学院、管理员。

**教师功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 科研课题
6. 软件著作权
7. 论文信息
8. 专利信息
9. 科研成果
10. 科研课题校审
11. 著作权校审
12. 论文校审
13. 专利校审
14. 科研成果校审


**学院功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 科研课题管理
6. 软件著作权管理
7. 论文信息管理
8. 专利信息管理
9. 科研成果管理
10. 科研课题校审管理
11. 著作权校审管理
12. 论文校审管理
13. 专利校审管理
14. 科研成果校审管理



**管理员功能：**
1. 登录
2. 首页
3. 个人中心（修改密码、个人信息）
4. 教师管理
5. 学院管理
6. 科研课题管理
7. 软件著作权管理
8. 论文信息管理
9. 专利信息管理
10. 科研成果管理
11. 科研课题校审管理
12. 著作权校审管理
13. 论文校审管理
14. 专利校审管理
15. 科研成果校审管理
16. 管理员管理
